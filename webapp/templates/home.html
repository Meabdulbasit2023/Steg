The pipeline

How a secret becomes invisible

1 · Binary → two's complement

Every byte of the secret is converted to binary, then transformed with S₂comp = NOT(S) + 1 — a reversible scramble that hides the original values.

2 · XOR against the cover

Each bit is XORed with the pixel it will occupy, tying the payload to the specific image so a raw read reveals nothing.

3 · Machine-learning placement

A Random Forest ranks every 8×8 block by texture and hides bits in the busiest areas first, where the human eye can't see the change.

4 · Verify & measure

An SVM attacker tries to detect the result; PSNR and MSE confirm quality. The secret is recovered bit-for-bit.

Project aim & objectives

What this system sets out to do

Aim: design an enhanced image-steganography framework that uses two's complement and machine learning for image hiding.

  1. Design the two's complement steganography framework.
  2. Implement it (this web application, built in Python).
  3. Evaluate performance with PSNR and MSE, and compare with traditional techniques (LSB, DCT).
  4. Deploy and recommend the best method by performance and accuracy.
