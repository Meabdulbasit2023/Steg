Embedding the first byte, bit by bit

Each 2's-complement bit is XORed with the most-significant bit (MSB) of the ML-chosen pixel it will live in, then written into that pixel's lowest bit. The MSB never changes, so the decoder can reverse the XOR exactly.

{% for r in trace.bits %} {% endfor %}
bit #S₂comp bitpixel slotpixel before MSBbit ⊕ MSBpixel after
{{ r.k }}{{ r.bit }}{{ r.slot }} {{ r.before }} = {{ r.before_bin }} {{ r.msb }} {{ r.bit }} ⊕ {{ r.msb }} = {{ r.xored }} {{ r.after }} = {{ r.after_bin }}

Decoding runs this in reverse: read the low bit, XOR with the same MSB, collect 8 bits, then apply 2's complement again (it is its own inverse: NOT(S₂comp) + 1 = S) to recover each original byte.
